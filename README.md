### Deploy to Vercel

1. Push this repo to GitHub.
2. Import the repo at [vercel.com/new](https://vercel.com/new) (Vercel auto-detects `dist` from `vercel.json`).
3. Or deploy from the CLI:

```bash
nvm use
npm install
npx vercel --prod
```
